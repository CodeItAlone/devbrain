# Contributing to DevBrain

We welcome contributions to DevBrain! 

## How to Contribute
1. Fork the repository and create your branch from `main`.
2. Ensure you have Node.js LTS (>=20) installed.
3. Install dependencies: `npm install`.
4. Make your changes and write tests where applicable.
5. Verify build, format, and tests:
   ```bash
   npm run build
   npm test
   ```
6. Submit a Pull Request.

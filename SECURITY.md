# Security Policy

## Reporting a Vulnerability

We take the security of DevBrain seriously. If you find a security vulnerability, please do not report it publicly via GitHub issues. Instead, report it directly to the maintainers.

Please send an email to the address specified in the security contacts (or open a private security draft on GitHub) with details on how to reproduce the issue.

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to release a security fix.
